##Herramientas utilizadas

* Framework KumbiaPHP (PHP 5.4)
* Plantilla adminLTE basada en Bootstrap
* Jquery
* MySql


---

##Comentarios de desarrollo
Fue desarrollado en el framework kumbiaPHP debido a que esta basado en MVC(Modelo Vista Controlador) y las
facilidades que este ofrece a la hora de desarrollar una aplicacion web.

En el FrondEnd fue usado la plantilla adminLTE la cual esta basada en bootstrap

##Instalación
Para correr la aplicacion web debe configurar los parametros de la base de datos en la siguiente direccion del
proyecto default/app/config/databases.ini en el proyecto le dejo el prueba.sql para importar la base de datos
